# Be Still and Know

> *"Be still, and know that I am God."* — Psalm 46:10

A free, private KJV Bible study progressive web app. No account. No server. No tracking. Works on any device — including offline.

**[→ Open the App](https://rainfirestorm.github.io/walk-on-water/)**

## Features

| Feature | Description |
|---|---|
| 📖 **Read the Bible** | Full KJV reader — all 66 books, every chapter, verse by verse |
| ✦ **Verse of the Day** | Fresh KJV verse every morning |
| 🔍 **Word Search** | Search any word across the entire Bible |
| ⭐ **Favorites** | Star verses and see them scroll across the top banner |
| 🖊 **Highlights** | Color-code verses in 5 colors |
| 🔖 **Bookmarks** | Save your reading position |
| 📝 **Notes & Insights** | Journal your thoughts on any verse |
| 🙏 **Prayer Journal** | Log prayers, mark them answered |
| 📖 **30-Day Reading Plan** | Read the whole Bible in 30 days with themes and reflections |
| 🕊 **Biblical Timeline** | From Creation to Revelation — every major event with dates |
| 🌿 **Family Tree** | Full genealogy from Adam to Jesus with biographical details |
| 🔊 **Read Aloud** | Text-to-speech with voice selection and speed control |
| ⇄ **Compare Translations** | KJV side-by-side with WEB and BBE |
| 📲 **Install on Phone** | PWA — add to home screen on iPhone or Android |
| 📊 **Reading Progress** | Track chapters, verses, and words read across the whole Bible |
| 🔥 **Reading Streak** | Daily streak tracker |

## Install on Your Phone

No App Store needed.

**iPhone / iPad**
1. Open the link in **Safari**
2. Tap the **Share** button
3. Tap **Add to Home Screen**

**Android**
1. Open the link in **Chrome**
2. Tap **Install App** when the banner appears

## Tech Stack

```
HTML5 · CSS3 · Vanilla JavaScript (ES6+)
No frameworks. No dependencies. No build step.
```

| Service | Purpose |
|---|---|
| [bible-api.com](https://bible-api.com) | KJV verse lookup (free, no key) |
| [bolls.life](https://bolls.life) | Full-Bible word search (free, no key) |
| Web Speech API | Read-aloud (browser-native) |
| SubtleCrypto API | AES-GCM localStorage encryption (browser-native) |
| GitHub Pages | Hosting (free) |

---

## Privacy

All your data — notes, prayers, highlights, bookmarks — is stored **only on your device**, encrypted with AES-GCM via the browser's built-in SubtleCrypto API. Nothing is sent to any server. No analytics. No tracking. No accounts.
